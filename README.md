# Laravel + React Project

This project uses Laravel for the API/Backend and React (via Vite) for the Frontend.

## Prerequisites
- **PHP** >= 8.2
- **Composer**
- **Node.js** >= 18
- **PNPM** (`npm install -g pnpm`)

## 🚀 Quick Start
Run the setup script to install dependencies and prepare the database, then start the application:

```bash
chmod +x setup.sh
./setup.sh
composer run start
